# 裂縫注膠補強計算 PWA

ACI Rap-1 混凝土構件裂縫注入環氧樹脂補強計算工具

## GitHub Pages 部署

1. 將所有檔案上傳至 repo **根目錄**（不需要子資料夾）
2. Settings → Pages → Source: `main` branch `/root`
3. 存取 `https://<username>.github.io/<repo>/`

## 安裝為 PWA App

- **Android**：Chrome → 選單 → 新增至主畫面
- **iOS**：Safari → 分享 → 加入主畫面
